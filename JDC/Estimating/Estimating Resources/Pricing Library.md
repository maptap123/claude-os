# JDC Pricing Library — 2026 Benchmarks

Distilled from 2026 estimates across kitchen, bath, addition, and dormer jobs. Use these as the **first reference for unit pricing**. The `JDC Cost Codes - Revised (Default).xlsx` file holds the canonical unit costs; this doc captures **patterns and gotchas** that aren't obvious from the code list alone.

Every code listed here is a builder cost (pre-markup). Apply 50% markup for client price.

## Pricing patterns & gotchas

- **Old-format estimates split many codes into Materials/Labor/Subcontract rows** (one cost code → up to 3 rows with "Materials" / "Labor" / "Subcontract" suffix on the Cost Code column). Pricing on each row reflects only that portion. The new BT format consolidates this.
- **Allowance lines** are written into the Description field, not into Quantity or Unit Cost. The Unit Cost is the line price; the allowance text just tells the client what's earmarked for fixtures.
- **Markup is uniform 50%** on every line in remodels. No exceptions seen across the 2026 sample.
- **Plans/Permits (01.0000.010)** is sometimes set to $0 in Total Cost (line zeroed) but with a Unit Cost of $500 or $1,200. Confirm with August whether to charge or absorb.
- **Pricing varies by job** for the same code (e.g., 07.0000. ranges $1.10–$3.35/SF in Crist vs Califf). This reflects different quantities/conditions, not arbitrary changes. When pricing a new line, prefer the value from the most recent comparable job rather than the cost-code-list value.
