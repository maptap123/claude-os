# New House Estimating — Current State & Gap

## What I observed in the Jobs folder

For **new builds** (not remodels), JDC has historically used a **lump-sum phase cost sheet**, not the line-item revised cost code system. Confirmed by these files:

- 2024 Cameron — `Cameron New House Cost Sheet.xls` — 5-column phase sheet
- 2026 Wagner — `Cost Sheet.Wagner.xlsx` — same phase sheet
- 2026 Wood — `Cost Sheet.Wood.xls` — same
- 2026 Masters — `House Cost Sheet.Masters.xls` — same
- 2026 Crossing at Tanners — `Cost Sheet.xlsx` — same
- 2026 Cotten — only photos, no estimate

## Phase cost sheet format

5 columns:
1. **PHASE** (e.g., Permits, Driveway, Excavation, Footers, Foundation, Waterproofing, Flatwork, Trenching, Septic, Tap-ins, Well, Framing, Framing package, Steel Beams, Floor package, Roof Trusses, Roof labor, Roof material, Brick mason, Brick material, Stone mason, Stone material, Siding labor, Ext. trim labor, Ext. trim/siding materials, Deck, Patio, Plumbing, Plumbing fixtures, Electric, Lighting allowance, HVAC, Drywall, Painting, Flooring, Trim Labor, Trim material, Stairs, Cabinets, Cabinet labor, Fireplace, Doors, Windows, Garage door, Insulation, Landscaping, Shower doors, Accessories, Appliances, Final Cleaning, TOTAL, Realtor Commission)
2. **CONTRACTOR** (sub or supplier name)
3. **CONTACT** (rep name)
4. **Main House** (lump sum builder cost)
5. **With Markup** (client price)

Markup observed:
- 2024 Cameron: 20% (1200/1000 = 1.20)
- 2026 Wagner: 25% (1250/1000 = 1.25)
- 2026 Wood: 25%
- 2026 Crossing at Tanners: not yet filled
- 2026 Masters: 20%

So new-house markup runs 20–25%, **not 50%** like remodels. This is the standard contractor practice — lower margin on a lump-sum new build vs. higher margin on remodel work where risk/complexity is higher.

## The gap August wants to close

Going forward, August wants new-house estimates built in the same line-item cost code system as remodels (using the revised codes), and exported in the new Buildertrend format. This means:

1. **Take a phase from the cost sheet** (e.g., "Foundation" with a $20,000 sub quote).
2. **Translate it into line-item rows** using the revised cost codes — e.g., 04.0620 (concrete wall, +LF), 04.4000 (continuous footings), 04.6310 (4" slab) — with quantities pulled from the plans.
3. **Validate the total against the sub quote.** If line-item math comes in lower than the sub's quote, the cost code rates may be stale for that scope and need a one-job override (price the line at the sub-quoted rate). If it comes in significantly higher, double-check quantities.
4. **Apply markup.** Whether to keep 20–25% on new builds or move to 50% across the board is an open decision August needs to make. **Flag this and ask** before exporting a new-build estimate.
5. **Allowances stay in the line description** the same way as remodels (lighting allowance, appliance allowance, etc.).
